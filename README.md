# BELAJAR GOLANG — Dari Dasar sampai Expert

![Go Gopher](assets/images/go-gopher-color.png)

Repositori ini berisi materi belajar Go bertahap dari **fundamental** sampai **expert**, fokus pada pemahaman konsep, praktik, dan kualitas engineering (testing, concurrency, performance, security).

## Jalur belajar

| Level | Folder | Fokus |
|---|---|---|
| 0 | `00-roadmap.md` | Peta belajar keseluruhan |
| 1 | `01-fundamental/` | Dasar bahasa Go |
| 2 | `02-intermediate/` | Package/module, testing, I/O, JSON |
| 3 | `03-advanced/` | Concurrency, context, memory model |
| 4 | `04-expert/` | Profiling, GC, PGO, fuzzing, coverage |
| 5 | `05-practice-capstone/` | Latihan bertingkat + capstone |

## Cara pakai

1. Ikuti urutan file dari atas ke bawah per level.
2. Kerjakan latihan kecil di setiap bab (disarankan membuat folder latihan lokal sendiri).
3. Setelah selesai 1 level, lanjut ke level berikutnya.
4. Gunakan `REFERENCES.md` untuk pendalaman resmi.

## Target pembelajaran

- Paham sintaks dan idiom Go.
- Mampu menulis kode concurrent yang aman.
- Mampu menguji, menganalisis performa, dan mengoptimasi aplikasi Go.
- Siap membuat service Go yang production-minded.
